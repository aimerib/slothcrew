Available Commands

These are the commands you can use to navigate the game.

look Examine your surroundings
inventory Check what you're carrying
take [item] Pick up an object
drop [item] Put down an object
use [item] Use or operate an item
talk to [person] Speak with someone
go [direction] Move north, south, east, west
help Show more commands
